 Como funciona?

 Digite no campo abaixo uma URL qualquer (Ex.: www.github.com). Você receberá uma URL encurtada. Se copiar e colar esta URL encurtada no browser, você será redirecionado para a página que tinha digitado ao gera-la.
Mas atenção! A URL encurtada só ficará válida durante 60 segundos!
